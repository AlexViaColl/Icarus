Learning Rust by making a game

# Requirements
- Rust >= 1.57, we recommend using [rustup](https://rustup.rs/).
- [Vulkan SDK 1.3](https://vulkan.lunarg.com/) + capable graphics card.

# Instructions
```console
# Run one of the binaries
cargo run --bin minesweeper

# Build everything
cargo build

# Run tests
cargo test
```
